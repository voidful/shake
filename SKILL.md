# Taiwan Hand-Shaken Drink Skill

This skill enables the LLM to become an expert in Taiwanese hand-shaken beverages. It includes massive structured data for 300 major drink brands and deep domain knowledge.

## How to use
When a user asks for a drink recommendation, check their preferences (brand, sugar, ice, dietary restrictions, caffeine). Query the internal database to provide accurate recommendations.
